# Calton — Ideas & FODA

## 1. Wizard

**Ideas**

1. **Event Brief Wizard** — multi-step: tipo evento → nro asistentes → fecha → presupuesto → email. Genera un brief descargable en PDF al final.
2. **Budget Estimator** — sliders interactivos (asistentes, días, catering, A/V). Output: rango estimado en tiempo real.
3. **Event Template Selector** — "¿qué querés lograr?" (team building / lanzamiento / conferencia) → recomienda paquete o combo de servicios.
4. **ROI Calculator** — inputás nro de asistentes + objetivo de negocio → output: métricas esperadas (NPS, engagement, retención).
5. **Venue Finder** — filtra por capacidad, ciudad, outdoor/indoor → muestra opciones del portfolio de Calton.

**FODA**

| | |
|---|---|
| **F** | Captura leads calificados; diferencia de la competencia; reduce fricción vs. formulario plano |
| **O** | Automatiza calificación de prospectos; permite segmentar propuestas; integrable con CRM |
| **D** | Lógica de negocio compleja; fácil de abandonar a mitad si hay demasiados pasos |
| **A** | Si las preguntas no están bien calibradas genera frustración; competidores pueden copiar rápido |

## 2. Features de Negocio

**Ideas**

1. **Case Studies / Portfolio** — sección filtrable por industria (fintech, retail, pharma). Cada caso: objetivo, solución, resultados.
2. **Testimonials con Logos** — carrusel de empresas clientes con quote + foto de contacto. Proof social fuerte.
3. **Comparador de Paquetes** — tabla sticky side-by-side (Starter / Pro / Custom). Destaca columna recomendada.
4. **FAQ Interactivo** — accordion con las objeciones más comunes. Reduce carga al equipo de ventas.
5. **Blog / Insights** — artículos cortos sobre tendencias en eventos corporativos. Alimenta SEO y posiciona a Calton como referente.

**FODA**

| | |
|---|---|
| **F** | Genera confianza real; comunica propuesta de valor de forma concreta |
| **O** | SEO orgánico con contenido; thought leadership; aumenta tiempo en sitio |
| **D** | Requiere contenido real (fotos, datos, testimonios); actualización continua |
| **A** | Contenido genérico destruye credibilidad; sin fotos reales pierde impacto |

---

## 3. Features AI

**Ideas**

1. **Chatbot de Captación** — responde FAQs, califica leads y agenda llamadas. Modelo liviano (Haiku) en sidebar flotante.
2. **Event Brief Generator** — el usuario describe su evento en lenguaje natural → AI genera un brief estructurado descargable.
3. **Concept Suggester** — inputás industria + objetivo + mood → AI sugiere 3 conceptos creativos con paleta de colores y nombre.
4. **Propuesta Personalizada** — tras el wizard, AI arma un deck/PDF con los servicios relevantes, pricing estimado y casos similares.
5. **Trend Analyzer** — sección dinámica que muestra tendencias actuales en eventos corporativos (alimentada por AI + scraping o feeds).

**FODA**

| | |
|---|---|
| **F** | Diferenciación fuerte en el mercado; "wow factor"; escala sin contratar más personal |
| **O** | Automatiza calificación de leads; posiciona a Calton como agencia tech-forward |
| **D** | Costo de APIs; latencia visible; output de AI puede ser incorrecto o genérico |
| **A** | Privacidad de datos del cliente; expectativas desalineadas si AI falla en vivo |

## 4. Estética

**Ideas**

1. **Custom Cursor** — cursor personalizado que cambia de forma/color según la sección o sobre elementos interactivos.
2. **Scroll-Triggered Cinematics** — texto que se divide/ensambla al hacer scroll, imágenes que "revelan" su contenido con clip-path.
3. **Video Background Inmersivo** — hero con video loop de eventos reales, con overlay semitransparente y texto encima.
4. **Page Transitions con Framer** — wipe/curtain entre secciones al navegar (aunque sea single page, en la nav overlay).
5. **Variable Typography** — headlines con variable fonts que se estiran/comprimen en hover o según scroll speed.

**FODA**

| | |
|---|---|
| **F** | Primera impresión fuerte; refleja la creatividad de la agencia; eleva percepción de marca |
| **O** | Viralidad / shareable; diferenciación visual clara sobre competidores |
| **D** | Performance impact (LCP, CLS); accesibilidad comprometida si se abusa |
| **A** | Modas visuales envejecen rápido; sobre-diseño puede distraer del mensaje de negocio |

---

## 5. Awwwards Improvements

**Ideas**

1. **Horizontal Scroll Section** — sección de servicios o portfolio con scroll horizontal tipo "carrusel editorial", frecuente en ganadores.
2. **WebGL / Three.js Hero** — partículas o geometría 3D reactiva al cursor. Alto impacto en jurado técnico.
3. **Editorial Grid Layout** — rompé la columna central: títulos que sangran fuera del grid, texto superpuesto sobre imágenes, asimetría controlada.
4. **Loading Screen con Identidad** — preloader con animación del logo + counter que dé tiempo a cargar assets pesados. Los jueces lo ven siempre.
5. **Noise / Grain Texture + Gradientes** — capa de grain sutil sobre secciones + gradientes mesh. Estética que puntúa alto hoy en Awwwards.

**FODA**

| | |
|---|---|
| **F** | Credencial pública de calidad; atrae clientes de alto valor que buscan agencias "premium" |
| **O** | PR gratuito; referral traffic desde Awwwards; portfolio de diseño para el equipo |
| **D** | Requiere pulido extremo (pixel-perfect, sin bugs en inspección); proceso lento |
| **A** | Jurado subjetivo; algunas mejoras para Awwwards pueden dañar conversión real (UX vs. wow) |
